## 이번 시간 학습 목표

이번 세션에서는 **더 풍부한 포켓몬 정보**를 표시하기 위해 앱을 확장합니다.

### 추가되는 기능
1. **타입 배지**: 포켓몬의 타입(불꽃, 물, 풀 등)을 색상 있는 배지로 표시
2. **능력치 게이지**: HP, Attack, Defense, Speed를 프로그레스 바로 시각화
3. **성능 최적화**: API 요청을 병렬 처리하여 로딩 속도 개선
